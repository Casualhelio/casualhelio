Website Text Editor

Edit any text on your site — in English, Mongolian, and Japanese — by clicking it directly on the page.

How it works

  1. Pick a page below. It opens in edit mode (a navy toolbar appears at the top).
  2. Click any text and type your change. Use the EN / MN / JA buttons in the toolbar to edit each language.
  3. Click 💾 Save. Your changes are written to a file called translations.js.
  4. Upload that one file to your web host (same place you publish the site) to make it live.

Choose a page to edit

Home → About → Our Companies → Investment → Contact → News page →

News articles themselves are edited in your Sanity CMS at nest-group.sanity.studio — this editor handles all the fixed wording around them.

💡 For one-click saving: open this through a local web address (http://localhost) in Chrome or Edge — then Save writes straight to translations.js. If you just double-click the file instead, Save will download the new translations.js; replace the one in your js/ folder with it and re-upload.
